¿Qué es nodejs?Cibergrafía
by Juan Fernando Vélez


A diferencia del uso tradicional de JavaScript en el navegador, Node.js permite que el código JavaScript se ejecute en el servidor, lo que significa que los desarrolladores pueden utilizar el mismo lenguaje de programación tanto en el lado del cliente como del servidor. Node.js utiliza un modelo de operaciones de entrada/salida no bloqueante y basado en eventos, lo que significa que puede manejar múltiples conexiones simultáneas sin bloquear el flujo del programa.

Node js, se basa en el principio de programación asíncrona, la cual es una técnica que permite a tu programa iniciar una tarea de larga duración y seguir respondiendo a otros eventos mientras esa tarea se ejecuta, en lugar de tener que esperar hasta que esa tarea haya terminado. Una vez que dicha tarea ha finalizado, tu programa presenta el resultado.

Muchas de las funciones proporcionadas por los navegadores, especialmente las más interesantes, pueden tardar mucho tiempo en ejecutarse. Es precisamente por eso por lo que esas funciones son asíncronas. Por ejemplo:

  • Realizar peticiones HTTP utilizando fetch()

  • Acceder a la cámara o micrófono de un usuario mediante getUserMedia()

  • Pedir a un usuario que seleccione archivos usando showOpenFilePicker() (en-US)

    • Por lo tanto, aunque no tenga que implementar sus propias funciones asíncronas muy a menudo, necesitará utilizarlas correctamente.

    Otras ventajas de node js

    Es de código abierto
    Al tratarse de un proyecto de código abierto, este se encuentra en constante actualización por la comunidad, además de tener su código fuente, publicado.
    Cross platform
    Node js puede ser ejecutado en una gran variedad de sistemas operativos y plataformas.
    Gratuito
    Node.js es una biblioteca de código abierto y gratuito, esto ha permitido que se encuentre ampliamente difundido y sea de preferencia por la comunidad de desarrolladores.
